# movie-app
A Search Movie app made for two days Hackathon Challenge.<br>

![](movie-app.gif)

## Project setup
```
* CD into folder movie-app
* Run npm install & npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Technologies

* VUE CLI, VUETIFY, AXIOS,  <br>
